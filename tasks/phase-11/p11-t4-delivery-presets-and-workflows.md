# [P11-T4] Delivery Presets And Workflows

## Status

Not started

## Phase

[Phase 11 - Format and Delivery Expansion](../../roadmap/phase-11-format-and-delivery-expansion.md)

## Outcome

Add export presets that map technical settings to common publishing and review workflows.

## Scope

- Define a small preset set for review, social, and archive-oriented deliveries.
- Connect presets to codec, geometry, and audio behavior choices.
- Surface preset intent clearly in the export UI.

## Implementation Notes

- Presets should reduce ambiguity, not hide critical technical behavior.
- Workflow labels need to explain whether they imply crop, pad, audio retention, or codec changes.

## Acceptance Criteria

- The export UI exposes a clear preset set tied to actual delivery workflows.
- Selecting a preset updates relevant geometry, codec, and audio settings coherently.
- The user can still inspect or override the preset plan before exporting.
