# [P10-T4] Export Throughput Profiling

## Status

Not started

## Phase

[Phase 10 - High-Resolution Preview and Performance](../../roadmap/phase-10-high-resolution-preview-and-performance.md)

## Outcome

Measure export throughput at larger rasters and establish practical support targets.

## Scope

- Profile export time and bottlenecks across a representative size matrix.
- Compare decode, CPU color evaluation, and encode cost.
- Define the first officially supported large-raster target.

## Implementation Notes

- This task should produce decision-ready measurements, not just instrumentation hooks.
- Results should separate source complexity from raster-size effects where practical.

## Acceptance Criteria

- Export throughput is measured across a documented size matrix.
- The team has a recommended first high-resolution support target.
- Major throughput bottlenecks are identified and recorded.
