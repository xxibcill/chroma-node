# P7-T5 Packaging And Release

## Status

Not started

## Phase

[Phase 07 - Hardening and Packaging](../../roadmap/phase-07-hardening-and-packaging.md)

## Outcome

The MVP can be packaged, launched outside the dev environment, and shipped with clear limitations.

## Scope

- Configure desktop packaging.
- Bundle or locate FFmpeg consistently.
- Add startup diagnostics in packaged builds.
- Create release notes.
- Document MVP-supported media and known limitations.

## Implementation Notes

- Package macOS first unless the target platform decision changes.
- Verify app paths differ between development and packaged environments.
- Keep release notes honest about Resolve-inspired behavior and non-goals.

## Acceptance Criteria

- Packaged app launches outside development tools.
- Import, grade, save, open, track, and export smoke test passes in packaged app.
- FFmpeg is available or a clear diagnostic is shown.
- MVP limitations are documented.
