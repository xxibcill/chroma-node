## Phase

## Outcome

The app is measured and tuned against the MVP performance targets.

## Scope

- Measure import time.
- Measure first-frame time.
- Measure playback fps with 1 and 3 nodes.
- Measure parameter latency.
- Measure scope update latency.
- Measure tracking and export speed.

## Implementation Notes

- Use an Apple M1-class machine as the reference target unless changed.
- Record clip metadata for every measurement.
- Optimize only after establishing baseline numbers.

## Acceptance Criteria

- Performance measurements are documented.
- Critical misses are either fixed or explicitly accepted.
- Viewer remains responsive during common grading operations.
- Memory usage stays below target for supported clips.
