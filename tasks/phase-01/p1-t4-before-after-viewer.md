# P1-T4 Before/After Viewer

## Status

Not started

## Phase

[Phase 01 - Import, Viewer, Playback](../../roadmap/phase-01-import-viewer-playback.md)

## Outcome

The user can compare original and graded output with original, graded, and adjustable split modes.

## Scope

- Add viewer mode state.
- Add original mode.
- Add graded mode.
- Add vertical split mode.
- Add adjustable split position.

## Implementation Notes

- Before grading exists, original and graded may be visually identical.
- Split mode should keep the left side original and right side graded.
- The split control should not affect export.

## Acceptance Criteria

- Viewer mode can toggle between original, graded, and split.
- Split position moves from 0% to 100%.
- Playback and scrub work in all viewer modes.
- Mode state is ready for project serialization.
